# 20秒＋10秒 ビープタイマー

GitHub Pages でそのまま公開できる、単一HTMLのタイマーページです。

## 動作
- 開始ボタンを押す
- 20秒後にビープ音
- さらに10秒後にビープ音
- これを8セット繰り返す

## 公開方法
1. GitHubで新しいリポジトリを作る
2. `index.html` をアップロードする
3. Repository の **Settings** → **Pages** を開く
4. **Build and deployment** で **Source = Deploy from a branch** を選ぶ
5. **Branch = main / (root)** を選んで保存
6. 数分待つと公開URLが発行される

## 補足
- スマホでもアクセス可能です
- 音を鳴らす都合上、最初の開始ボタンはユーザーが押す必要があります
