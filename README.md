# 🔥 HIIT-LOG

GitHub Pages でそのまま公開できる、単一 HTML の HIIT タイマーページです。

## 動作
- `開始` を押すと 5 秒カウントダウンが始まり、その後すぐにワークアウト開始ブザーが鳴ります
- ワークアウト中の状態表示は `待機中` / `運動中` / `休憩中` の 3 つです
- ワークアウトの合図音は、短い電子音ではなく「ブーッ！」と鳴る持続系のブザー音です
- 開始から停止・完了までは、対応ブラウザーで `Screen Wake Lock` を要求し、非対応環境では `NoSleep.js` で画面スリープ防止を補います
- `停止・リセット` を押さずに最後まで完了すると、その実施日時がブラウザーに保存されます
- 保存された履歴はページ下部に新しい順で表示されます

## 補足
- 音を鳴らす都合上、最初の開始ボタンはユーザー操作が必要です
- PC / スマホのどちらからでも利用できます
- 実施履歴はそのブラウザーの `localStorage` に保存されます
- 画面スリープ防止はブラウザーや省電力状態によって効かない場合があります
